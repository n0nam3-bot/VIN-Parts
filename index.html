VIN Parts Finder

Decode a VIN with the public NHTSA API, then generate manufacturer-focused part search links and category shortcuts for GitHub Pages.

The VIN decoder uses NHTSA vPIC. It does not bypass OEM catalog restrictions.

OEM / Parts Search Shortcuts

Deep OEM catalog extraction is not generally available from GitHub Pages alone. This page creates useful lookup links from VIN-decoded vehicle data, and you can extend it later with your own parts database or catalog JSON.

How it works

1) VIN is decoded through the NHTSA API.
2) Make/model/year/body details are displayed when available.
3) Category links are generated for the decoded vehicle family.
4) For exact OEM fitment, you would still need a catalog source or export.
